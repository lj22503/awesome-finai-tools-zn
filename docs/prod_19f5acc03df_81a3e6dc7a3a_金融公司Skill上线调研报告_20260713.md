## 四、对 Invest Brain / Work Buddy 的关键启示

### 4.1 你的场景一：用户先用了 Work Buddy，觉得数据有缺

**缺口来源分析：**
- Work Buddy 本身是一个 Agent 框架，不提供金融数据
- 用户如果只用 Work Buddy 内置的通用能力做投资分析，会遇到：
  - 行情数据不及时/不准确（大模型幻觉）
  - 缺少结构化金融数据（财务指标、估值、基金持仓等）
  - 缺少专业分析框架（杜邦分析、归因分析等）

**可接入的外部 Skill 填补方案：**

| 数据缺口 | 可选外部 Skill | 覆盖能力 |
|---------|--------------|---------|
| 行情/财报数据 | 万得 AIFin Market (MCP) | A股/港股/美股全量数据 |
| 基金数据 | 易方达指数直通车 Skills | ETF + 场外指数基金 |
| 个股分析 | 东方财富妙想 Skills | 综合诊股、财务查询 |
| 宏观数据 | 东方财富妙想宏观查询 | GDP/CPI/PMI等 |
| 选股筛选 | 国泰海通智能选股 Skill | 自然语言选股 |

### 4.2 你的场景二：用户已有其他公司 Skill，想建自己的信息库

这正是 Invest Brain 的核心定位——**「镜子」+「纪律锚点」**。关键策略：

1. **定位差异化**：不跟券商/数据商抢"数据提供者"角色，而是做"数据之上的决策层"
   - 他们提供行情 → 你提供"我为什么在这个价格买"
   - 他们提供财报 → 你提供"这个财报符合我的投资框架吗"
   - 他们提供选股 → 你提供"我上次选这只股票时犯了什么错"

2. **引流路径设计**：
   - 用户在天天基金/东方财富 Skill 里查到数据 → 不知道如何决策
   - Invest Brain 作为"下一站"：记录想法、关联历史、挖掘行为模式
   - 价值主张：**"数据他们给，纪律我来帮"**

3. **技术可行性（已验证）**：
   - 易方达已对接 WorkBuddy，说明 WorkBuddy 生态支持金融 Skill 互操作
   - 万得通过 MCP 协议开放数据，Invest Brain 的 MCP Server（已有35个工具）可以与之并存
   - 用户在同一 WorkBuddy 实例中安装 Invest Brain Skill + 天天基金 Skill + 万得 MCP，AI 自动编排调用

## 五、风险提示

- 银行/保险机构的 Skill 多为内部使用，短期内不会对外部 Agent 开放
- 券商 Skill 的合规边界仍在摸索中——它们能开放到什么程度、是否允许"被组合调用"，是持续需要关注的政策变量
- 万得/同花顺/东财的 MCP/Skill 的收费模式尚未完全明朗，接入成本需要评估
