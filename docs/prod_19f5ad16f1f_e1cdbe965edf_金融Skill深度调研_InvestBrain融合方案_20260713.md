### 2.2 场景一：用户先用了 Work Buddy，觉得数据有缺

**缺口 → 填补方案对照表**：

| 用户说 | 缺什么 | 装上哪个外部 Skill | 具体怎么装 |
|--------|-------|-------------------|-----------|
| "帮我看看这只基金怎么样" | 基金诊断 | 东方财富妙想综合诊基 | IMA/Coze 一键安装，财富号自带鉴权 |
| "查一下这只ETF的持仓" | ETF深度数据 | 易方达指数直通车 | 小程序领Key → WorkBuddy一键安装 |
| "对比茅台和五粮液的财报" | 财务对比 | 广发证券财务对比 | gf.com.cn 注册领Key → OpenClaw安装 |
| "最近市场在涨什么" | 热点/龙虎榜 | 国泰海通市场热榜 或 广发龙虎榜 | 各自 App 领资格 |
| "帮我按条件筛股票" | 选股引擎 | 华泰条件选股 或 东财智能选股 | AI涨乐/东财 App 获取 |
| "这只股票的机构研报怎么说" | 研报提取 | 国泰海通研报搜索 | 灵犀 Skills 安装 |
| "宏观环境对市场的影响" | 宏观数据 | 东财宏观查询 + 万得AIFin | 东财免费 / 万得 MCP |
| "做一次完整的估值分析" | 全维度金融数据 | 万得 AIFin Market | MCP 接入 WorkBuddy |

**然后 Invest Brain 接手**：
- 用户查完数据 → "这些数据说明了什么？我该怎么做？"
- Invest Brain 不是再给一份数据，而是：
  - 记录用户的判断："我当时认为茅台合理估值在 XX"
  - 关联历史："你上次在类似情况下买了，后来涨了 15%"
  - 行为审计："你最近 3 次决策都是追高买的，这和你的'安全边际'原则冲突"
